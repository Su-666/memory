## 基本使用

### 保存记忆

直接用自然语言告诉暖暖要记住的内容：

- "帮我记住 WiFi 密码是 12345678"
- "记一下：张三的电话是 13800138000"
- "提醒我明天下午3点开会"
- "收藏这篇文章的链接"
- "写下来：我的生日是 1990年1月1日"

暖暖会自动提取标题和摘要，保存到数据库并归档到文件保险库。

**待保存确认**：如果只说"帮我记"而没有具体内容，暖暖会追问。此时补充内容即可保存，或者说"算了"取消。

### 搜索记忆

- "帮我找一下 WiFi 密码"
- "查一下张三的联系方式"
- "之前记过的那个会议是什么时候"
- "我的手机号是多少"

搜索会自动扩展关键词（如"手机号"同时搜索"电话"、"号码"），并按相关性排序返回结果。

### 闲聊对话

暖暖可以像朋友一样聊天：

- "你好"
- "今天心情不错"
- "给我讲个故事"
- "帮我写一首诗"

聊天模式下暖暖会保持知性温和的语气，不卖萌、不客套。

### 图片理解

通过以下方式上传图片：

1. 点击输入框左侧的附件按钮
2. 直接拖拽图片到输入区域
3. 粘贴剪贴板中的图片

暖暖会自动识别图片内容，生成描述文字和标签，并提取图片中的文字（OCR）。识别结果会作为记忆保存，后续可通过关键词搜索到。

### 语音交互

点击麦克风按钮进入语音模式：

- **单次录音**：点击麦克风按钮开始录音，再次点击停止
- **连续对话**：进入语音模式后，暖暖会持续监听，说完自动识别并回复
- **打断回复**：在暖暖说话时点击麦克风可打断当前回复

语音模式下暖暖的回复会自动朗读。

### 文件上传

支持上传的文件类型：

| 类型 | 格式 |
|------|------|
| 图片 | jpg, jpeg, png, gif, webp, bmp |
| 文档 | pdf, doc, docx, txt, md |
| 音频 | mp3, wav, m4a, aac, flac, ogg |
| 视频 | mp4, mov, avi, mkv |

最多同时上传 10 个文件，总大小限制 50MB。上传的文件会保存到保险库目录。

### 快捷命令

| 命令 | 说明 |
|------|------|
| `/搜 关键词` 或 `/search 关键词` | 直接搜索，跳过意图判断 |
| `/聊` 或 `/chat` | 直接进入聊天模式 |

## 管理员后台

访问 `/admin` 路径，输入管理员密码登录。

- 本地：`http://127.0.0.1:5000/admin`

需要在环境变量（`.env`）中设置 `ADMIN_KEY`。

### 功能

- **数据概览** — 总记忆数、文件记忆数、标签数量、最近更新时间
- **记忆管理** — 浏览全部记忆，支持搜索、排序、分页
- **记忆详情** — 点击任意记忆查看完整内容、标签、文件路径、时间戳
- **删除记忆** — 在详情弹窗中删除，操作不可撤销
- **导出 JSON** — 导出全部记忆为 JSON 文件
- **下载数据库** — 下载 SQLite 数据库文件备份

## 桌面端使用

桌面端基于 PySide6 + PyQt-Fluent-Widgets 构建，采用 Fluent 设计风格，通过 API 连接本地服务器。

### 启动

```bash
python pyqt_local_agent.py
```

### 功能特性

- **Fluent 设计风格** — 圆角卡片、渐变背景、统一色彩体系
- **暗色模式** — 一键切换亮色/暗色主题（Ctrl+T），设置自动保存
- **语音对话** — 支持语音输入和语音回复，连续对话模式
- **文件拖拽** — 直接拖拽文件到输入区域上传
- **自动补全** — 输入框支持 `/搜`、`/search`、`帮我记住` 等命令补全
- **统计面板** — 查看记忆总数、热门标签、存储信息
- **图片预览** — 搜索结果中的图片可点击放大查看
- **断线重连** — 网络中断后自动重连，指数退避策略
- **打字机效果** — 助手回复逐字显示动画
- **快捷键** — Ctrl+T 切换主题，Ctrl+L 清空对话

### 配置

桌面端连接本地服务器 `http://127.0.0.1:5000`，需先启动 Web 服务再打开桌面端。

如需使用语音功能，需在 `.env` 中配置百度语音密钥（`BAIDU_APP_ID`、`BAIDU_API_KEY`、`BAIDU_SECRET_KEY`）。

### 打包为 EXE

```bash
python build_exe.py
```

生成的 `dist/记忆助手.exe` 为单文件可执行程序，无需安装 Python 环境即可运行。

## API 接口

所有接口的基础 URL 为本地服务地址 `http://127.0.0.1:5000`。

请求和响应均为 JSON 格式，需要设置 `Content-Type: application/json`。

### 对话与意图

#### POST `/api/chat`

主聊天接口，自动判断意图（保存/搜索/聊天）。

**请求体：**
```json
{
  "text": "帮我记住 WiFi 密码是 12345678"
}
```

**响应（保存意图）：**
```json
{
  "type": "assistant",
  "text": "记好啦~ 以后忘了随时问我呀",
  "saved": true
}
```

**响应（搜索意图）：**
```json
{
  "type": "assistant",
  "text": "找到啦~ WiFi密码：12345678",
  "results": [
    {
      "entity_type": "memory",
      "entity_id": 1,
      "title": "WiFi密码",
      "summary": "WiFi密码是 12345678",
      "time": "2025-01-01 12:00:00"
    }
  ]
}
```

**响应（待确认保存）：**
```json
{
  "type": "assistant",
  "text": "好呀，具体是什么内容呢？说给我听听～",
  "pending_save": "帮我记"
}
```

#### POST `/api/chat/confirm_save`

确认或取消待保存内容。

**请求体：**
```json
{
  "text": "好的",
  "pending_text": "帮我记"
}
```

#### POST `/api/clear`

清除当前客户端的聊天历史。通过 `X-Client-Id` 请求头或 IP 地址识别客户端。

### 记忆管理

#### GET `/api/memories`

获取最近 30 条记忆列表。

#### GET `/api/memories/recent?limit=N`

获取最近记忆，`limit` 最大 50。

#### GET `/api/memory/<id>`

获取单条记忆详情。

#### PUT `/api/memory/<id>`

更新记忆内容。

**请求体：**
```json
{
  "content": "更新后的记忆内容"
}
```

#### DELETE `/api/memory/<id>`

删除指定记忆。

#### GET `/api/tags`

获取所有标签列表（缓存 300 秒）。

#### GET `/api/stats`

获取统计信息（缓存 60 秒）。

**响应：**
```json
{
  "total_memories": 42,
  "total_files": 15,
  "tags": ["密码", "电话", "工作"],
  "top_tags": [["密码", 5], ["电话", 3]],
  "last_updated": "2025-01-01 12:00:00",
  "storage_size": 1048576,
  "storage_formatted": "1.0 MB"
}
```

### 搜索

#### POST `/api/search`

简单搜索。

**请求体：**
```json
{
  "query": "WiFi 密码"
}
```

**响应：**
```json
{
  "results": [
    {
      "entity_type": "memory",
      "entity_id": 1,
      "title": "WiFi密码",
      "summary": "WiFi密码是 12345678",
      "body_snippet": "...",
      "time": "2025-01-01 12:00:00",
      "score": 15.5
    }
  ]
}
```

#### POST `/api/memories/search`

高级搜索，支持标签和日期范围过滤。

**请求体：**
```json
{
  "query": "密码",
  "tags": ["WiFi"],
  "date_from": "2025-01-01",
  "date_to": "2025-12-31",
  "limit": 20
}
```

### 文件上传接口

#### POST `/api/upload`

上传文件（multipart/form-data）。

**表单字段：**
- `files` — 文件（支持多个）
- `caption` — 文件说明文字（可选）

**curl 示例：**
```bash
curl -X POST http://127.0.0.1:5000/api/upload \
  -F "files=@photo.jpg" \
  -F "caption=这是我的猫"
```

### 语音

#### POST `/api/speech_recognize`

语音识别（ASR），上传 WAV 音频文件。

**表单字段：**
- `audio` — WAV 音频文件

#### POST `/api/speech_synthesize`

语音合成（TTS），返回 base64 编码的音频。

**请求体：**
```json
{
  "text": "你好，我是暖暖"
}
```

**响应：**
```json
{
  "audio": "UklGRi...",
  "format": "wav"
}
```

#### POST `/api/voice_dialogue`

完整语音对话流程：语音识别 → 意图判断 → 回复生成 → 语音合成。

**表单字段：**
- `audio` — WAV 音频文件
- `skip_tts` — 设为 `true` 跳过语音合成（仅返回文字）
- `pending_save_text` — 待保存文本（用于确认保存流程）

### 系统信息

#### GET `/api/health`

健康检查。

**响应：**
```json
{
  "status": "ok"
}
```

#### GET `/api/version`

版本信息。

#### GET `/api/diag`

诊断端点，检查 LLM 配置和连通性。

#### GET `/api/vault/path`

获取保险库文件系统路径。

#### GET `/api/file/image?path=<path>`

预览保险库中的图片文件。

### 管理员接口

所有管理员接口需要通过 `X-Admin-Key` 请求头或 `?key=` 查询参数提供管理员密码。

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/admin/verify?key=` | 验证密码是否正确 |
| GET | `/api/admin/data?key=` | 返回全部记忆和统计数据 |
| DELETE | `/api/admin/memory/<id>?key=` | 删除指定记忆 |
| GET | `/api/admin/export?key=` | 导出全部记忆为 JSON 文件 |
| GET | `/api/admin/backup?key=` | 下载 SQLite 数据库文件 |
