# 暖暖记忆助手 - Web版

## 🚀 新功能特性

### ✨ 核心优化
- **性能监控**: 自动跟踪API响应时间和错误率
- **智能缓存**: GET请求结果自动缓存，提升响应速度
- **速率限制**: 防止恶意请求，保护服务器资源
- **错误处理**: 更详细的错误信息和调试日志

### 🎨 用户体验
- **暗色模式**: 一键切换亮色/暗色主题
- **统计面板**: 查看记忆数量、热门标签等统计信息
- **响应式设计**: 完美适配手机和桌面设备
- **加载状态**: 更直观的加载和处理状态显示

### 🔍 搜索增强
- **高级搜索**: 支持标签、日期范围等多条件搜索
- **标签系统**: 为记忆添加和管理标签
- **实时过滤**: 快速找到相关记忆内容

### 🛡️ 安全与稳定性
- **输入验证**: 防止恶意输入和注入攻击
- **文件安全**: 严格的文件类型和大小检查
- **异常处理**: 完善的错误恢复机制

## 📊 API端点

### 基础功能
- `GET /api/health` - 健康检查
- `POST /api/chat` - 发送聊天消息
- `POST /api/search` - 搜索记忆
- `POST /api/save` - 保存记忆

### 语音功能
- `POST /api/speech_recognize` - 语音识别
- `POST /api/speech_synthesize` - 语音合成

### 记忆管理
- `GET /api/memories/recent` - 获取最近记忆
- `GET /api/memories/search` - 高级搜索
- `GET /api/tags` - 获取所有标签
- `GET /api/stats` - 获取统计信息
- `GET /api/memory/<id>` - 获取记忆详情
- `PUT /api/memory/<id>` - 更新记忆
- `DELETE /api/memory/<id>` - 删除记忆

### 文件处理
- `POST /api/upload` - 文件上传
- `GET /api/file/image` - 图片预览

## 🔧 配置要求

### 环境变量
```bash
# 智谱AI API
ZHIPU_API_KEY=your_api_key

# 百度语音API
BAIDU_APP_ID=your_app_id

BAIDU_SECRET_KEY=your_secret_key

# 服务器配置
```

## 📱 使用说明

### 基本操作
1. **聊天对话**: 在输入框输入文字或点击麦克风录音
2. **保存记忆**: 说"帮我记住..."或"记一下..."
3. **搜索记忆**: 说"帮我找..."或"查一下..."
4. **查看统计**: 点击右上角"📊 统计"按钮

### 高级功能
- **标签管理**: 为记忆添加标签便于分类
- **主题切换**: 点击月亮图标切换暗色模式
- **文件上传**: 拖拽文件到聊天区域
- **图片预览**: 点击记忆中的图片链接

## 🔍 故障排除

### 语音功能无法使用
- 检查百度API配置是否正确
- 查看Railway日志中的错误信息
- 确认网络连接正常

### 页面加载慢
- 检查浏览器缓存
- 查看网络连接
- 尝试刷新页面

### 搜索无结果
- 检查搜索关键词是否正确
- 尝试使用更简单的关键词
- 查看记忆是否已保存
